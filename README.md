# PixelForge Storefront UI

Premium solo-creator digital products storefront built with Next.js App Router, Tailwind CSS, Framer Motion, Lucide icons, and Zustand.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Routes

- `/`
- `/products`
- `/products/[slug]`
- `/freebies`
- `/cart`
- `/checkout`
- `/confirmation`
- `/account`
- `/about`
- `/search?q=...`
- `/404`

## Validation done

- `npm run lint` passed.
- `npm run build` passed.
